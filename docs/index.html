Setup & Run Locally

You need an Anthropic API key. No deployment costs — runs entirely on your machine.

Clone the repository

git clone https://github.com/PritamSonawane7798/india-stock-ai.git
cd india-stock-ai

Create a virtual environment and install dependencies

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt

Add your Anthropic API key

cp .env.example .env
# Edit .env and set:
ANTHROPIC_API_KEY=sk-ant-...

Get your API key at console.anthropic.com. The app uses Claude Sonnet — check current pricing there.

Launch the Streamlit app

streamlit run src/app.py

Opens at http://localhost:8501

Or run the demo script (CLI)

python examples/demo_analysis.py
